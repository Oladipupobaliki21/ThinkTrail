ThinkTrail
----

📌 Project Overview

A modern E-Learning web application built with React and Tailwind CSS, designed to help users explore courses, watch lessons, and track their progress.

----
🚀 Features


- Course Exploration:** Browse and filter courses by category.  
- Search Functionality:** Quickly find courses by keywords.  
- Video Player:** Watch lessons directly in the browser.  
- Mark as Completed:** Track which lessons you’ve completed.  
- Persistent Progress:** Lesson completion is saved using `localStorage`.  
- Dynamic Navigation:** Navigate between lessons and back to the home page anytime.  
- Responsive Design:** Works on desktop and mobile devices.
